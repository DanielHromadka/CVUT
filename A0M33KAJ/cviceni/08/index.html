HR - File upload

  • Úkol: Napište aplikaci, která odešle na server soubor. 
  • var file = e.dataTransfer.files[0]; // z eventy drop

var xhr = new XMLHttpRequest()
xhr.open("POST", "/upload", true);
xhr.setRequestHeader("X-FILENAME", file.name);
xhr.send(file);
