Koncept this

  • Kdy this nabývá své hodnoty?
  • Úkol: Vyrobte třídu, jejíž konstruktor vyrobí div a na něj navěsí na click fci, která vypíše hodnotu promněné this.message
  • http://jsfiddle.net/koko/tjm4U/
  • var Visual = function () {
	this._message = 'ahoj svete!';
    this._dom = document.createElement('div');
    this._dom.style.height = "50px";
    this._dom.style.backgroundColor = "gray";
    document.body.appendChild(this._dom);
    this._dom.addEventListener('click', this._show,true);
}
Visual.prototype._show = function () {
	alert(this._message)
}

var v = new Visual();

Prototypy - modulo záporného čísla

  • Úkol: Upravte celočíselné dělení, tak aby vracelo kladné hodnoty zbytku i pro záporné hodnoty vstupu
  • http://jsfiddle.net/koko/tjm4U/
  • Number.prototype.mod = function (divisor) {
	return ((this % divisor) + divisor) % divisor
}

Prototypy - náhodný výběr prvku z pole

  • Úkol: Upravte objekt Array, tak aby měl metodu rnd, ktré vrátí náhodný prvek z pole.
  • http://jsfiddle.net/koko/tjm4U/
  • Array.prototype.rnd = function () {
	var rIndex = Math.floor((this.length)*Math.random());
	return this[rIndex];
}

Apply - Inspekční middleware

  • Úkol:Napište fci, která modifikuje fce ostatní a to tak, že budou vypisovat do console, info o tom, že jsou volány
  • http://jsfiddle.net/koko/tjm4U/
  • function log (name,owner) {
    var owner = owner || window;
    var f = owner[name];
    owner[name] = function () {
        console.log('Running function > ' + name);
        return f.apply(this, arguments);
    }
}
