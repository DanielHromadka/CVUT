SVG - základy - teorie. Opáčko.

  • Co znamená SVG?
  • Scalable Vector Graphics
  • Založeno na XML, stylovatelné pomocí CSS, možno vyrábět pomocí JS
  • Jak přidáme SVG do HTML5 stránky?
  • Tag <svg>
  • Jaké základní tvary a konstrukce můžeme využít?
  • <rect>, <circle>, <ellipse>, <line>, <polyline>, <polygon>
  • <g>, gradients + patterns

SVG - základ

  • Úkol: Vytvořte ve stránce místo pro "kreslení".
  • <svg width="300" height="300"></svg>

  • Úkol II: Obarvěte za pomocí CSS a zařiďte, že při najetí nad oko se oko schová

Drag&Drop

  • Umožňuje manipulovat s objekty ve stránce i přetahovat externí soubory do stránky
  • Přesunovatelný prvek - atribut draggable="draggable"
  • Události: dragstart, dragover, drop
  • Vlastnost dataTransfer - obsahuje metadata
  • Pokud má být nad prvkem tažení ukončitelné, je nutné událost "preventovat" - protože v defaultu není "dropnutí" dovoleno
  • Pokud událost zpracujeme, voláme preventDefault
  • Úkol: Napište JS aplikaci, která umožní uživateli přetáhnout do stránky soubor a stránka vypíše meta informace o souboru (typ, velikost, umístění)
