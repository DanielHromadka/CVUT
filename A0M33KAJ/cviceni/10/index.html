Canvas - práce s pixely

  • Jak se dostanu k pixelům obrázku?
  • Úkol I: Do canvasu vykreslete barevný obrázek, který trasformujete na černobílý a pak inverzní (nabídněte k uložení). 
  • var imageData = ctx.getImageData(0, 0, img.width, img.height);
var data = imageData.data;
for(var i = 0; i < data.length; i += 4) {
  var p = Math.round((data[i] + data [i+1] + data[i+2])/3)
  data[i] = p; //cervena
  data[i + 1] = p; // zelena
  data[i + 2] = p; // modra
}
ctx.putImageData(imageData,0,0);
