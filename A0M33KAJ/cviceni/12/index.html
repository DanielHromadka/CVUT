Aplikace

  • Bude zaznamenávat na uživatelovo přání polohu a případně vypisovat uložené souřadnice
  • Dvě "stránky"
  • Použijeme GEO API a History API

GEO API

  • Jak se dostanu k aktulání poloze zařízení?
  • navigator.geolocation
  • navigator.geolocation.getCurrentPosition(callback, error)
  • Callback funkce má jako parametr objekt s vlastností coords

Local Storage API

  • Jak,co a kolik MB dat můžu uložit do LS?
  • 5MB
  • window.localStorage: key, getItem, setItem, removeItem, length
  • ukládám dvojci klíč - hodnota (řetězce)

History API

  • Jakmůžu zapisovat do historie prohlížeče?
  • window.history.pushState(stateObj,title,url)
    window.history.replaceState(stateObj,title,url)
  • Jak můžu reagovat na kliknutí na šipky zpět a dopředu v browseru?
  • window.onpopstate = function(event) { /** něco vykonej **/ }
