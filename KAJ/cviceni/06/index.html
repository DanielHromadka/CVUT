Animations - opakování

  • K čemu slouží?

@keyframes posun {
    0%, 100% { right: 0px;   top: 100px; }
    25%      { right: 200px; top: 100px; }
    50%      { right: 200px; top: 300px; }
    75%      { right: 0px;   top: 300px; }
}
